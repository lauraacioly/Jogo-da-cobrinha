# 🐍 Jogo da Cobrinha

Jogo clássico da cobrinha desenvolvido em Python com a biblioteca Pygame. Projeto acadêmico com tema visual personalizado.

## 📌 Funcionalidades

- Movimentação da cobrinha com as setas do teclado
- Sistema de pontuação ao coletar itens
- Tela de game over ao colidir
- Elementos visuais personalizados (imagens de fundo e sprites)

## 🚀 Como rodar

1. Tenha o Python instalado na máquina
2. Instale o Pygame:
   ```bash
   pip install pygame
   ```
3. Clone o repositório:
   ```bash
   git clone https://github.com/lauraacioly/Jogo-da-cobrinha.git
   ```
4. Execute o jogo:
   ```bash
   python cobrinha.py
   ```

## 💡 Aprendizados

- Lógica de jogos com loop principal e detecção de colisão
- Manipulação de eventos de teclado
- Renderização de elementos visuais com Pygame
- Estruturação de um projeto interativo em Python
